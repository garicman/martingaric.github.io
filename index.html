> Projets_

[01] Projet Arduino - Domotique

Système de contrôle domotique basé sur Arduino pour la gestion de l'éclairage, de la température et de la sécurité. Utilisation de capteurs DHT22, relais et modules RF 433MHz.

Technologies : Arduino, C++, MQTT, Node-RED

[02] Station Météo Connectée

Station météorologique autonome avec ESP8266, envoyant les données (température, humidité, pression) vers un serveur web pour visualisation en temps réel.

Technologies : ESP8266, BME280, Python, Flask

[03] Robot Autonome

Robot mobile évitant les obstacles, utilisant des capteurs ultrasoniques et un algorithme de navigation simple. Projet éducatif pour l'apprentissage de la robotique.

Technologies : Arduino Uno, Moteurs DC, HC-SR04

[04] Application Web de Gestion

Application web responsive pour la gestion de projets personnels avec authentification, base de données et interface utilisateur moderne.

Technologies : HTML/CSS/JavaScript, Node.js, MongoDB
